# Code conventions — MoTDex

MoTDex is a **Nuxt 4 static site**, not a component library. Conventions live in two skills — read both before coding:

1. **[`nuxt-app`](../nuxt-app/SKILL.md)** — `app/` layout, Vue/TS, components, composables, `baseURL`-safe paths.
2. **[`static-deploy`](../static-deploy/SKILL.md)** — GitHub Pages constraints, static JSON, no runtime server.

When `docs/content/conventions.md` exists, it becomes the human-facing source of truth; these skills stay as agent pointers.
